# Система управления рецептами

В этом проекте я создал систему с различными действиями по управлению рецептами
Она содержит классы Ingredient, Recipe, ShoppingList и DietaryRecipe, с их помощью я могу добавлять ингредиенты, собирать рецепты и составлять список покупок, также можно определять сколько продуктов потребуется для определенного количества людей.

## Описание структуры кода

Запуск кода:

\`\`\`
python classes.py
\`\`\`

Запуск тестов с помощью pytest:

\`\`\`
pip install pytest
pytest
\`\`\`
